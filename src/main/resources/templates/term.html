1 – Do tratamento e dados coletados
1.1 Dos dados coletados

Nosso site coleta e utiliza alguns dados pessoais, voluntariamente fornecidos pelo usuário, quais sejam:

  • O número do CRM do usuário;
  • A especialidade médica do usuário;
  • O tempo de experiência em sua área de atuação.
1.2 Do tratamento dos dados

Os dados coletados serão tratados pelo Controlador/Operador com a finalidade de uso não econômico e acadêmico, sendo o uso fora das características indicadas condicionado a prévia autorização do usuário.

Entende-se por tratamento a coleta, produção, recepção, classificação, utilização, acesso, reprodução, transmissão, distribuição, processamento, arquivamento, armazenamento, eliminação, avaliação ou controle da informação, modificação, comunicação, transferência, difusão ou extração, entre outros, dos dados fornecidos pelo usuário.

1.3 Do armazenamento dos dados

Os dados fornecidos pelo usuário serão utilizados e armazenados em banco de dados privado durante o tempo necessário para as finalidades elencadas na presente Política de Privacidade, considerando os direitos dos titulares dos dados e dos controladores.

Os dados fornecidos serão mantidos pelo Controlador/Operador pelo tempo que o mesmo entender necessário. O Controlador/Operador se reserva no direito de, sem prévio aviso ou consentimento, tratar os dados fornecidos.

Essas ferramentas são aplicadas levando em consideração a natureza dos dados pessoais coletados, o contexto e a finalidade do tratamento.

O Controlador/Operador não se responsabiliza por danos decorrentes de culpa, negligência, imperícia ou imprudência do usuário.
